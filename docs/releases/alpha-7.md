# Mesh Utility Alpha 7

Alpha 7 is now available.

This release focuses on clearer node discovery behavior, accurate coverage mapping, and more reliable data sync.

---

## Live App

Use the web app:
[https://mesh-utility.org/](https://mesh-utility.org)

---

## What's Improved

### Node Discovery

* Discover requests now behave more predictably during real-world use
* Radio activity is easier to understand while scanning
* Improved handling of advert data for better contact and name detection

---

### Coverage / Dead Zones

* Coverage maps now stay accurate as new scans come in
* Areas previously marked as dead zones are automatically corrected when valid data exists
* Reduces false gaps in coverage

---

### Sync / Data Reliability

* Large scan histories load more reliably
* Data updates are more consistent across sessions
* Improved handling of high-volume scan environments

---

### Storage & Identity

* Devices are identified more consistently across scans and uploads
* Reduces duplicate or mismatched data

---

### Performance

* Lower storage overhead during extended use
* More stable behavior during long scanning sessions

---

## Why This Matters

Alpha 7 makes everyday use more predictable and accurate:

* Discovery works more consistently with live radios
* Coverage maps better reflect real-world conditions
* Sync is more reliable, even with large datasets
